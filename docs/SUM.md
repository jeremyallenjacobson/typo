# Software User Manual (SUM)

## τύπο: Reader Guide

### Navigation

| Action | Desktop | Mobile |
|--------|---------|--------|
| Next page | Right arrow, Down arrow, Space, Page Down | Swipe left, or tap right edge |
| Previous page | Left arrow, Up arrow, Page Up | Swipe right, or tap left edge |
| First page | Home | — |
| Last page | End | — |
| Rapid flip | Hold any navigation key | Hold tap on left/right edge |

### Display

The display is always dark — Apple Books Night mode (#b0b0b0 text on #121212 background). There is no light mode and no toggle.

### Direct Links

Append `#` and a page number to the URL to link directly to a page. For example:

```
jeremyjacobson.dev/yatp/#3
```

This opens page 3. Share these links with others to point them to a specific page.
